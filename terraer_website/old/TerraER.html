TerraER

 
 

O TerraER 2.0 é é um programa gratuito, de código aberto (disponibilizado no SourceForge), que facilita o aprendizado de modelagem de dados. A ferramenta é inicialmente voltada ao meio acadêmico, mais especificamente no auxílio ao aprendizado de disciplinas de modelagem de banco de dados.


A ferramenta está em constante desenvolvimento e é disponível como projeto de código aberto.


O software funciona como um editor e possui algumas funcionalidades básicas:

  1. •Construção do modelo de entidade e relacionamento (ER);

  2. •Interface simples;

  3. •Facilidade de aprendizado;

  4. •Geração de arquivos em formato XML;

  5. •Exportação para área de transferência;

  6. •Impressão.


A ferramenta TerraER é bastante funcional e facilita o aprendizado de modelagem de dados, visto que os modelos são fáceis de serem criados e modificados.
